## 담당 역할

음식점 지역별/종류별 조회, 식당 검색, 가게 정보 상세 조회 -> 👩‍💻 최윤경
음식점별/등급별 리뷰 조회, 리뷰 등록 -> 👩‍💻 권수민
내 리뷰 모아보기/삭제 -> 🧑‍💻 양도열
